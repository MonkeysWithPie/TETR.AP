# Caution!

If you open this folder in a code editor, it will throw errors about imports not resolving.

These files will not compile into an apworld in this directory. You'll need to create a copy of this folder in the `Archipelago/worlds` folder with a name of "tetr_ap".

See [Archipelago Docs](https://github.com/ArchipelagoMW/Archipelago/blob/main/docs/adding%20games.md#world) for more information.
