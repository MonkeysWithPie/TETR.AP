# TETR.AP

TETR.AP is an unofficial TETR.IO Archipelago mod.

## Randomization

TETR.AP randomizes each individual tarot card unlock within the game, as well as Achievement checks that are created for each achievement tier of each modset.  

## Installation & Usage

### Client

WARNING: **DO NOT** report bugs to osk whilst using the TETR.AP client!! Issues should be reported to this GitHub repository instead. Posting issues in the TETR.IO discord with the TETR.AP client installed will likely get you banned.

> [!Note]
> TETR.IO Desktop is not supported right now! We're working on it.

#### Browser

1. Download or copy the contents of the `client.min.js` file.
2. Open your browser of choice and create a bookmark.
3. Type "javascript:" into the URL section of the bookmark and then paste the contents of the client file.  
It should look something like this: `javascript:!async function(){"use strict";const e="[TETR.AP]",...`
4. Save the bookmark (the name of it does not matter).
5. Open TETR.IO.
6. Click the bookmark you just created while staying in the TETR.IO tab. A menu should appear in the top right.
7. You're good to go!

### YAML creation

1. Download the APWorld file.
2. Open or download the [Archipelago Launcher](https://github.com/ArchipelagoMW/Archipelago/releases).
3. Open the APWorld file. You should see a popup from Archipelago telling you that the APWorld was installed.
4. Open the "Options Creator" and find the TETR.AP entry.
5. Create your options as you like. Option descriptions can be seen by hovering the option name.
6. Press export options.

### Game generation

See also: [Archipelago Setup Tutorial](https://archipelago.gg/tutorial/Archipelago/setup_en#generating-a-game)

1. Download the APWorld file.
2. Open or download the [Archipelago Launcher](https://github.com/ArchipelagoMW/Archipelago/releases).
3. Open the APWorld file. You should see a popup from Archipelago telling you that the APWorld was installed.
4. Scroll down to the bottom and open "Browse Files".
5. Open the Players folder and place all the .yaml files you wish to use in the folder (NOT the Templates folder).
6. Go back to the Archipelago Launcher and open "Generate". It may take a while to generate, depending on how many games you have.
7. In the same folder where the "Players" folder was, there will now be an "output" folder.  
You can either host the room yourself (which this guide won't cover) or [host it via the Archipelago website](https://archipelago.gg/uploads).
